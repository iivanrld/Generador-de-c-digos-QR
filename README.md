## 🔌 API REST

### Generar QR

<details>
<summary><b>POST /generate</b></summary>

**Endpoint:** `http://localhost:5000/generate`

**Request:**
```json
{
  "type": "url",
  "url": "https://ejemplo.com",
  "color": "#00ff9f",
  "bgcolor": "#0a0a12"
}
```

**Response (200):**
```json
{
  "image": "data:image/png;base64,...",
  "data": "https://ejemplo.com"
}
```

**Errores posibles:**
- ❌ `400` — Campos obligatorios faltantes
- ❌ `500` — Error en generación

</details>

### Descargar QR

<details>
<summary><b>POST /download</b></summary>

**Endpoint:** `http://localhost:5000/download`

**Request:**
```json
{
  "image": "data:image/png;base64,..."
}
```

**Response:** 
- Archivo PNG descargado directamente

</details>

## 🎨 Personalización Avanzada

### Cambiar Tema de Colores

Edita `templates/index.html` (líneas CSS):

```css
:root {
  --bg:       #07070f;    /* Fondo oscuro */
  --bg2:      #0d0d1a;    /* Fondo secundario */
  --glow:     #00ff9f;    /* Verde neón principal */
  --glow2:    #00c8ff;    /* Cian secundario */
  --red:      #ff3c6e;    /* Rojo acentos */
  --text:     #c8ffe8;    /* Texto */
}
```

### Temas Pre-configurados

| Tema | Colores | Vibe |
|------|---------|------|
| 🟢 **Neon Green** | `#00ff9f` + `#07070f` | Moderno & Pro |
| 🔵 **Cyber Blue** | `#00c8ff` + `#0a0a12` | Frío & Tech |
| 🌸 **Magenta** | `#ff3c6e` + `#1a0a14` | Vibrante & Bold |
| 💜 **Purple Dream** | `#c71585` + `#0d0d1a` | Místico & Elegante |
