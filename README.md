# QR Generator ◼

Generador de códigos QR con interfaz web **cyberpunk/neon** animada.
Desarrollado con **Python + Flask**.

## Inicio rápido

```bash
pip install -r requirements.txt
python app.py
```

El navegador se abre automáticamente en `http://localhost:5000`.

## Modos

| Modo | Contenido |
|------|-----------|
| 🔗 URL | La cámara abre el enlace directamente (no busca en Google) |
| 📄 Texto | Cualquier cadena de texto libre |
| 👤 Contacto | vCard 3.0 — la cámara ofrece guardar el contacto |

## Atajos

| Acción | Tecla |
|--------|-------|
| Generar QR | `Ctrl + Enter` |
